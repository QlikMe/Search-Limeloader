Welcome to Limeloader - Limelight Index Loader
A Jupyter Notebook Python App that builds Elastic Search indexes from CSV files.

Getting Started

  1. Install Jupyter on your Mac or PC. The Anaconda package is highly recommended.
  2. Clone/unpack/copy this project to a folder on your machine.
  3. Click File Open... in Jupyter (launched via localhost:8888 for most developers) and select Limelighter.ipynb
  4. Run the cells below as required to transform CSV files into an Elastic Search index.
